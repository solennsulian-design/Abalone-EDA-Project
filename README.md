# Abalone Age Prediction - EDA Project

## Objective
Predict the age of abalones (number of rings) from physical measurements to avoid manual counting under a microscope.

## Dataset
- **Source**: [UCI Abalone Dataset](https://archive.ics.uci.edu/ml/datasets/abalone)
- **Features**: Sex, Length, Diameter, Height, Whole weight, Shucked weight, Viscera weight, Shell weight
- **Target**: Rings (age indicator)

## Tools
- **Language**: Python (Google Colab)
- **Libraries**: `pandas`, `numpy`, `matplotlib`, `seaborn`, `scikit-learn`

## Project Files
- [Abalone_EDA.ipynb](Abalone_EDA.ipynb) - Main notebook with analysis
- [PRESENTATION.md](PRESENTATION.md) - Project summary
- [flowchart.png](flowchart.png) - Process diagram (optional)

## How to Run
1. Open [Abalone_EDA.ipynb](Abalone_EDA.ipynb) in [Google Colab](https://colab.research.google.com/).
2. Download the dataset from the UCI link above.
3. Run all cells in order.
